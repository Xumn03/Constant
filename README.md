# Constant

Constant is a cross-platform modeling and animation app.

## What changed from Constant to Blender and Maya
| Constant | Blender | Maya |
| -------- | ----- | ----------- |
| Beautiful, fully customizable | A bit ugly, fully customizable | Beautiful, non customizable  |
| Professional, open-source and free      | Hobbyist, open source and free|  Professional, Paid   |
| Very Lightweight, still really realistic    | A bit heavy, with 2 rendering modes| Heaver, pretty realistic. |

## Features
* Animation like video editing
* Use your camera to animate
* Big Face viewer
